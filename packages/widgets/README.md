# @modyra/widgets

Headless widget controllers and universal interaction/accessibility contract for Modyra.

## Scope

- Controller headless e state machine.
- Intent/command contract.
- State/view contract semantico.
- ID deterministic policy.
- Contract fixtures.
- Testing kit di conformità.

## Non-scope

- AST DOM generale.
- Virtual DOM proprietario.
- Gestione nodi/portals/teleport nel core.
- Accesso diretto al DOM dal controller.

See `.modyra/adr/0001-universal-widget-contract.md` for the architectural
rationale and tier policy.
